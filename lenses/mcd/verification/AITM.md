# MCD Lens: AITM (AI-Directed Content) Verification

## General: Any AITM Atom

1. **What AI systems were exposed to this package?** Identify every AI coding assistant, agent, or automated review tool that may have ingested the package's source, IDE plugins, CI-integrated assistants, and any agentic pipelines consuming dependency code. `[MCD]`

2. **What actions did those AI systems take during the exposure window?** Review agent logs, tool call histories, and outputs generated by AI assistants while the package was installed. Look for anomalous file reads, unexpected network calls, credential lookups, or generated code not reflecting developer intent. `[MCD]`

3. **Does the content reference specific infrastructure, credential patterns, or internal naming conventions?** Generic content suggests opportunistic targeting. Specific references suggest prior knowledge of the target environment. `[lens-neutral]`

4. **When was the AITM content introduced?** Present from first version = likely purpose-built delivery mechanism. Appearing in subsequent version of established package = potential supply chain compromise. `[lens-neutral]`

## AITM.INJECT

5. **Is the payload functional against current LLM context windows?** Does it reference specific agent frameworks (LangChain, Claude tool use syntax, AutoGPT) or exploit model-specific behaviors? Model-specific targeting indicates sophistication. `[lens-neutral]`

6. **Are other packages from the same author similarly affected?** AITM campaigns frequently distribute payloads across multiple packages. `[MCD]`

## AITM.TOOL

7. **Does the tool definition appear in any AI agent's active tool registry?** A definition present in the repository but never loaded into an agent runtime is lower urgency than one actively registered and callable. `[lens-neutral]`

8. **Does the tool schema request parameters beyond its stated function?** Particularly: environment variables, file paths outside the project, or authentication tokens. `[lens-neutral]`

## AITM.INVISIBLE

9. **What specific encoding technique is used?** Zero-width characters, homoglyphs, RTL overrides, tag characters, or adversarial token sequences each have different detection methods and effectiveness against different models. `[lens-neutral]`

## Cross-Cutting

10. **What is the organizational policy for AI agent tool approval?** If the package reached an AI agent's context without passing through any approval gate, the finding is also evidence of a process gap that extends beyond this specific package. `[lens-neutral]`
